Question-1:   What are the different ways to select an element in the DOM?

There are many way of selecting an element from html document. Among them, some common ways are getElementsByClassName(), getElementById(), getElementsByTagName(), querySelector(), querySelectorAll().

Question-2:   What is the difference between innerHTML, innerText, and textContent ?

While innerHTML return all the element inside an element inluding tag and text, innerText return only the value inside of an element. textContent is likely an innerText, however,it provides the all text including comments.

Question-3:   What is event delegation in the DOM?

Sometimes it becomes irriatting to write more and more addEventListener to control elements, so, event delegation can make it more easier by adding addEventListener to parentNode of any elements.

Question-4:   What is event bubbling in the DOM?

Browser does not know the exact location when any user click on a certain button. So, what browser does to find its loaction is propagate from parentNode to childNode until it finds where user click. Afterwards, it comeback to its origin. And this way is called event bubbling in DOM
